01 · The question engine

Questions have a skeleton.

Most errors happen because students memorize isolated questions instead of understanding the engine. The engine below builds most WH questions in present simple and present progressive.

WH gives the information target. The auxiliary carries the grammar.

If the student controls this sequence, they can create hundreds of questions without memorizing each one separately.

Step 1WH

What / Where / When / Who / Why / How

Step 2Auxiliary

do / does / am / is / are

Step 3Subject

I / you / she / they / the class

Step 4Main verb

base verb or verb-ing

Step 5Complement?

time, place, object, reason

Core rule: in present simple, does steals the -s from the main verb. Correct: Where does she work? Incorrect: Where does she works?

Present Simple

Use it for habits, routines, facts and general preferences.

WH + do/does + subject + base verb?

Present Progressive

Use it for actions happening now or temporary situations.

WH + am/is/are + subject + verb-ing?

Short Answers

Repeat the auxiliary. The answer mirrors the question.

05 · Class protocol

How to train WH questions.

A good WH exercise is not random guessing. It follows a sequence: identify the answer type, select WH, select auxiliary, build the question, then answer naturally.

01

Find the missing information

Answer: “I live in Cali.” Missing information: place. Therefore: Where?

02

Choose the time lens

Routine uses do/does. Now uses am/is/are + -ing.

03

Build the skeleton

WH + auxiliary + subject + verb. Do not move the main verb before the subject.

04

Answer and extend

Give a short answer first, then add one detail to make it conversational.

08 · Teacher guide

Run it as a live class.

This section is designed for the teacher. It gives a simple class route that keeps the page pedagogical, not just decorative.

Opening · 5 min

Ask students: “What information do you need when you ask a question?” Write: thing, place, time, person, reason, method.

Formula · 10 min

Use the skeleton: WH + auxiliary + subject + verb. Drill three examples slowly, then naturally.

WH Stations · 15 min

Each student chooses one WH word, explains what it asks for, reads one example and creates one personal question.

Builder + Quiz · 15 min

Use the Build Lab first. Then quiz. Correct errors by naming the broken part: WH, auxiliary, subject, verb or complement.
